# HR Automation Platform (InsightHR)

### Overview

Welcome to the InsightHR workshop! This serverless application helps businesses, especially in IT and related fields, customize and automate their HR evaluation and management processes. The application not only visualizes performance data but also provides a flexible platform for each company to apply their unique set of KPIs and evaluation formulas, enabling accurate and fair HR decisions.

### What You'll Learn

In this workshop, you will learn how to build a serverless HR automation platform using AWS services including Lambda, DynamoDB, Cognito, Bedrock, and more. You'll understand how to create a customizable performance evaluation system that adapts to any organization's unique needs.
